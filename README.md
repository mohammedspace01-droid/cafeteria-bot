# 🎧 Telegram Support Desk Bot

A lightweight, efficient customer support bot that bridges the gap between users and your admin team directly inside Telegram, transforming a Supergroup into a centralized Help Desk.

### ✨ Key Features
- **Centralized Ticketing:** Compiles user inquiries (text, media, voice) and forwards them to a designated Admin Group.
- **Seamless Replies:** Admins simply "Reply" to the forwarded message in the group, and the bot delivers the answer privately to the user.
- **Live Status Tracking:** Automatically tags inquiries as 🟢 Answered or 🟡 Pending.
- **Admin Dashboard:** Quick stats via the `/dashboard` command.
- **Lightweight:** Uses a local JSON file for session and memory management—no complex database required.

## 🌍 النسخة العربية (Arabic Description)

بوت دعم فني ذكي لإدارة استفسارات المستخدمين وتحويل تليجرام إلى نظام تذاكر (Ticketing System) متكامل.

### ✨ المشكلة والحل
إدارة استفسارات العملاء أو الطلاب عبر الخاص (DMs) عملية فوضوية. **هذا البوت يقوم بمركزية كل الاستفسارات في "جروب أدمن" واحد، مما يسهل الرد الجماعي وتنظيم العمل.**

### 🚀 المميزات الأساسية
- **مركزية الاستفسارات:** استقبال رسائل المستخدمين (نص، صور، ملفات صوتية) وتجميعها في تذكرة واحدة داخل جروب الإدارة.
- **الرد المباشر:** بمجرد عمل Reply من أي أدمن على رسالة البوت داخل الجروب، يتم إرسال الرد فوراً للمستخدم.
- **تتبع الحالة (Live Status):** تحديث تلقائي لحالة التذكرة (🟢 تم الرد / 🟡 لم يتم الرد).
- **لوحة تحكم (Dashboard):** عرض إحصائيات سريعة لعدد الاستفسارات المكتملة والمعلقة.
